# Sistem Manajemen Menu Makanan

Aplikasi web sederhana untuk mengelola data menu makanan menggunakan
HTML, CSS (Tailwind), dan JavaScript dengan penyimpanan data berbasis
Local Storage browser.

## Fitur
- Menambahkan data menu makanan
- Menampilkan daftar menu
- Mengedit data menu
- Menghapus data menu
- Menyimpan data secara permanen menggunakan Local Storage

## Teknologi yang Digunakan
- HTML5
- Tailwind CSS (CDN)
- JavaScript
- Local Storage Browser

## Cara Menjalankan Aplikasi
1. Clone atau download repository ini
2. Buka file `index.html` menggunakan browser
3. Tambahkan data menu melalui form yang tersedia

## Penyimpanan Data
Data menu disimpan menggunakan Local Storage pada browser sehingga
data tetap tersimpan meskipun halaman di-refresh, selama cache browser
tidak dihapus.

## Struktur Folder
menu-management/
├── index.html
├── assets/
│ ├── css/
│ └── js/


## Catatan
Aplikasi ini dibuat sebagai tugas pembelajaran untuk memahami konsep
manajemen data sisi klien menggunakan JavaScript dan Local Storage.
